# 民体动作智能分析系统使用指南

更新日期：2026-04-29

本文档面向 Windows `.exe` 交付版本，适用于教师、学生、实验记录人员和现场演示人员。

重点说明：

- 如何部署与启动 `动作分析工作台.exe`
- 如何生成标准模板
- 如何分析测试视频
- 如何使用前测 / 后测下拉框
- 如何查看实时建议与科研导出
- 如何排查“光流兜底”真实原因

## 1. 适用对象

- 民体课程教师
- 八段锦、健身气功、太极指导人员
- 学生训练记录人员
- 科研实验数据采集与整理人员
- 需要分发 `.exe` 给终端用户的维护人员

## 2. 程序形式

当前项目已编译为 Windows 桌面程序，主执行文件为：

```text
dist/动作分析工作台.exe
```

注意：

- 不建议只拷贝单个 `.exe`
- 建议保留完整目录结构后再运行
- 程序依赖 `weights/`、`fonts/`、`templates/`、`outputs/`

## 3. 推荐目录结构

```text
动作分析工作台/
├─ 动作分析工作台.exe
├─ weights/
├─ fonts/
├─ templates/
├─ outputs/
└─ 使用指南.md
```

目录说明：

- `weights/`：姿态识别权重
- `fonts/`：中文字体资源
- `templates/`：模板与模板库
- `outputs/`：分析结果输出目录

## 4. 首次运行前检查

请逐项确认：

1. 操作系统为 Windows 10 或 Windows 11
2. 程序目录已完整解压
3. 程序目录具备写权限
4. `weights/` 中至少存在一个可用姿态权重文件
5. `fonts/` 中字体文件存在
6. 如需摄像头实时分析，设备已接入并可被系统识别

推荐部署目录示例：

```text
D:\动作分析工作台\
```

## 5. 启动方式

双击运行：

```text
动作分析工作台.exe
```

首次启动稍慢通常属于正常初始化过程。

## 6. 主界面说明

### 6.1 左侧区域

左侧用于输入与控制，主要包括：

- 模板导入
- 模板生成
- 测试对象列表
- 前测 / 后测下拉框
- 测试视频导入
- 模板库选择
- 摄像头扫描与选择
- 动作描述输入

### 6.2 中间区域

中间区域用于画面预览与可视化，包括：

- 标准模板画面
- 测试视频画面
- 骨架叠加对比
- 实时状态摘要

### 6.3 右侧区域

右侧区域用于结果判读与回看，包括：

- 综合评定
- 指标差异
- 问题定位
- 调整建议
- 实时动作详细优化建议
- 数据导出
- 过程日志

## 7. 推荐使用流程

1. 启动程序
2. 先生成标准模板
3. 再导入测试视频
4. 在左侧明确选择“前测 / 后测 / 未标记”
5. 选择模板基线后开始分析
6. 查看右侧问题定位与建议流
7. 导出结果文件

## 8. 标准模板生成教程

### 8.1 推荐素材

建议使用：

- 单人标准动作视频
- 固定机位视频
- 全身完整入镜视频

不建议使用：

- 多人同时入镜视频
- 人物频繁出画视频
- 遮挡严重的视频

### 8.2 操作步骤

1. 进入模板区域
2. 导入标准动作视频
3. 填写模板名称
4. 填写动作描述
5. 点击“生成标准模板基线”
6. 等待分析完成

### 8.3 动作描述建议

建议至少包含：

- 动作名称
- 节奏要求
- 躯干要求
- 双臂要求
- 下盘要求
- 定势要求

示例：

```text
八段锦标准动作，节奏舒缓，要求躯干中正，双臂匀称舒展，重心稳定，定势时保持沉肩坠肘与气沉丹田。
```

## 9. 测试视频分析教程

### 9.1 操作步骤

1. 导入测试视频
2. 在测试对象列表中选择当前对象
3. 选择模板基线
4. 明确选择“前测 / 后测 / 未标记”
5. 点击“开始分析”
6. 等待分析结束

### 9.2 前测 / 后测下拉框说明

系统虽然会根据文件名自动推断，但建议你始终手工确认：

- `前测`
- `后测`
- `未标记`

原因：

- 汇总总表以该字段为准
- 改善量 Excel 需要准确的前后测标签

### 9.3 分析过程中可看到的内容

状态栏会持续显示：

- 进度百分比
- ETA 预计剩余时间
- 当前动作阶段
- 姿态后端状态

中间区域会显示：

- 模板画面
- 测试画面
- 骨架叠加对比

右侧区域会同步输出：

- 问题定位
- 调整建议
- 实时动作详细优化建议

## 11. 摄像头实时分析教程

### 11.1 操作步骤

1. 点击扫描摄像头
2. 选择设备
3. 如需模板对照，先加载模板
4. 点击开始分析
5. 如需录像，开启实时录制
6. 停止后查看结果

### 11.2 摄像头摆位建议

- 保证单人全身完整入镜
- 镜头尽量固定
- 保持适当距离
- 避免逆光
- 避免多人干扰

## 14. 前后测结果判读

### 14.1 身体稳定性参数

- 数值越小越好
- 改善量建议按：

```text
前测 Stability_CV - 后测 Stability_CV
```

### 14.2 动作流畅性参数

- `Transition_Acc_Peak` 越小越好
- `Transition_Jerk_Mean` 越小越好

### 14.3 身体调控能力参数

- `Control_Ratio` 已支持直接导出
- 反映定势维持与动作转换之间的张力调控差异

比值公式：

```text
Ratio = Σ过渡期位移幅度 / Σ定势期位移幅度
```

## 15. 常见问题排查

### 15.1 双击 exe 没反应

请检查：

- 是否完整解压
- 是否缺少资源目录
- 是否被安全软件拦截
- 是否位于无写权限目录

### 15.2 没有骨架，只显示光流兜底

请直接看界面中的真实提示原因，常见情况包括：

- 未安装 `Ultralytics`
- 权重文件不存在
- 权重路径错误
- 模型加载失败
- 推理失败

### 15.3 中文显示异常

请检查：

- `fonts/` 中字体文件是否存在
- 打开导出文件的软件是否支持 UTF-8

### 15.4 摄像头无法启动

请检查：

- 设备连接
- 权限设置
- 是否被其他软件占用
- 是否选择了错误的设备编号
