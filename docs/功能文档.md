# 民体动作智能分析系统功能文档

更新日期：2026-04-29

## 1. 系统功能定位

本系统是一套面向教学、训练评估和科研实验的动作智能分析工具，核心特点包括：

- 本地运行
- 标准模板对比
- 实时建议反馈
- 中文专业教学建议
- 科研指标留存
- Windows `.exe` 可交付

## 2. 功能清单

### 2.1 模板生成

支持从标准动作视频生成模板基线，用于后续测试视频对比分析。

输出包括：

- 模板基线 JSON
- 模板逐帧关键点
- 模板阶段划分
- 模板摘要与建议

### 2.2 测试视频分析

支持对测试视频执行：

- 姿态识别
- 阶段划分
- 问题定位
- 评分生成
- 模板对比
- 实时建议回传
- 结构化导出

### 2.3 摄像头实时分析

支持：

- 摄像头扫描
- 设备切换
- 实时分析
- 实时建议反馈
- 实时录制与回填

### 2.4 右侧栏分析区

右侧栏当前包括：

- 综合评定
- 指标差异
- 问题定位
- 调整建议
- 实时动作详细优化建议
- 数据导出
- 过程日志

## 3. 判断标准

### 3.1 姿态质量评分

姿态质量分综合：

- 姿态置信度
- 骨架稳定度
- 关键点覆盖率
- 可跟踪关键点比例
- 跟踪模式奖励

### 3.2 衔接评分

衔接评分综合：

- 运动信号平滑度
- 骨架位移平滑度
- 左右平衡稳定度
- 抖动抑制效果
- 跟踪稳定性

### 3.3 动作问题识别

当前重点识别项：

1. 躯干倾斜过大
2. 双臂对称误差过大
3. 模板框架偏差过大
4. 控制力不足或细微抖动
5. 定势不稳
6. 动作衔接不顺
7. 阶段切换异常

### 3.4 动作阶段

最终标准阶段包括：

- 准备
- 起势
- 过渡
- 定势
- 发力
- 收势

实时模式下可输出：

- 定势候选
- 过渡候选
- 发力候选

### 3.5 中文状态词统一

系统中所有状态词已统一为中文输出：

- 完美
- 极佳
- 良好
- 一般
- 注意
- 偏差
- 较差
- 缺失

## 4. 实时建议功能

### 4.1 触发规则

当前主要规则：

- `躯干倾斜 > 15°`
- `双臂对称误差 > 10`
- `template_distance > 0.5`
- `骨架稳定度偏低`
- `关节抖动明显`

### 4.2 输出内容

实时建议会组织为：

- 当前动作 / 当前阶段
- 问题表现
- 可能原因
- 调整方法
- 训练重点

### 4.3 实时建议流能力

右侧“实时动作详细优化建议”具备：

- 追加式日志流
- 相同内容去重
- 时间戳
- 自动滚动
- 可折叠
- 上下文保留

## 5. 光流兜底功能

当姿态模型不可用或局部推理失败时，系统可进入光流兜底模式，继续输出：

- 基础运动强度
- 左右平衡
- 候选阶段
- 基础实时提示

同时，GUI 会直接显示真实失败原因：

- 未安装 `Ultralytics`
- 权重文件不存在
- 模型加载失败
- 推理失败

## 6. 同源视频完美匹配

若检测到：

- 测试视频路径与模板源路径一致
- 或平均模板距离极低

系统将强制覆盖：

- 综合得分 `100.0`
- 分项得分 `100.0`
- 偏差值 `0.0`

并在摘要中追加：

```text
检测到基准视频自校准，动作完美匹配。
```

## 7. 科研指标功能

### 7.1 已有 5 项抑制类指标

1. 肢体晃动度
2. 躯干稳定度
3. 规范偏差
4. 角度稳定性
5. 无效小动作频次

### 7.2 新增 3 项运动生物力学指标

#### 身体稳定性参数 `Body Stability CV`

- 基于颈-髋连线动态偏移角度序列
- 计算 `CV = std / mean`
- 数值越小越稳定

#### 动作流畅性参数 `Action Fluency`

- 仅取 `过渡` 阶段
- 基于髋中心轨迹计算：
  - 过渡期加速度峰值
  - 过渡期急动度均值

#### 身体调控能力参数 `Body Control Ratio`

- 取 `定势` 末尾 10 帧
- 取 `过渡` 中间 10 帧
- 分别计算位移幅度
- 比值公式：

```text
Ratio = Σ过渡期位移幅度 / Σ定势期位移幅度
```

结论：

- `Control_Ratio` 当前已经支持导出
- 可以直接用于实验后测留存

### 7.3 前后测改善量

建议按以下口径做实验汇总：

- `前测 Stability_CV - 后测 Stability_CV`
- `前测 Transition_Acc_Peak - 后测 Transition_Acc_Peak`
- `前测 Transition_Jerk_Mean - 后测 Transition_Jerk_Mean`
- `前测 Control_Ratio - 后测 Control_Ratio`
